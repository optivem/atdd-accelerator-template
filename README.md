# ATDD Accelerator Template

## System Repositories

Acceptance Tests are independent of System Architecture, it doesn't matter whether your System Architecture is a Monolith, or Frontend & Microservice Backend, or Frontend & Microservice Backend. Thus, because it's irrelevant, for purposes of the ATDD Accelerator Program, we're providing you with the Monolith Templates, but feel free to choose any other Architecture if you choose to. It won't affect the Acceptance Tests.

- [ATDD Accelerator Template - Monolith (Java)](https://github.com/optivem/atdd-accelerator-template-monolith-java)
- [ATDD Accelerator Template - Monolith (.NET)](https://github.com/optivem/atdd-accelerator-template-monolith-dotnet)
- [ATDD Accelerator Template - Monolith (TypeScript)](https://github.com/optivem/atdd-accelerator-template-monolith-typescript)
